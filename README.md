# ArbiLink Partners - Лендинг

Профессиональный лендинг для компании ArbiLink Partners, специализирующейся на валютном арбитраже.

## Структура проекта

```
siteair/
├── index.html      # Основная HTML структура
├── styles.css      # Стили и оформление
├── script.js       # JavaScript для интерактивности
└── README.md       # Документация
```

## Особенности

- **Темная тема** с золотыми акцентами
- **Адаптивный дизайн** для всех устройств
- **Плавные анимации** при прокрутке
- **Современный UI/UX** дизайн
- **Оптимизированная производительность**

## Секции лендинга

1. **Hero Section** - Главная секция с брендингом и статистикой
2. **Partnership** - Информация о партнерстве с банками
3. **How We Work** - Описание работы компании
4. **Why Profitable** - Преимущества сотрудничества
5. **Guarantees** - Гарантии компании
6. **Payment Conditions** - Условия оплаты
7. **Contact** - Контактная информация и социальные сети
8. **Footer** - Подвал сайта

## Запуск проекта

1. Откройте файл `index.html` в браузере
2. Или используйте локальный сервер:

```bash
# С помощью Python
python -m http.server 8000

# С помощью Node.js (если установлен http-server)
npx http-server
```

Затем откройте в браузере: `http://localhost:8000`

## Настройка контактов

Для добавления реальных ссылок на социальные сети, отредактируйте файл `index.html` и замените `href="#"` на реальные ссылки в секции контактов:

```html
<a href="https://t.me/your_telegram" class="contact-button">
<a href="https://wa.me/your_whatsapp1" class="contact-button">
<a href="https://wa.me/your_whatsapp2" class="contact-button">
<a href="https://facebook.com/your_page" class="contact-button">
```

## Цветовая схема

- **Фон**: `#0A0A14` (темно-синий)
- **Карточки**: `#1A1A2E` (темно-серый)
- **Акцент**: `#FFD700` (золотой)
- **Текст**: `#FFFFFF` (белый)

## Шрифты

- **Заголовки**: Playfair Display (serif)
- **Основной текст**: Montserrat (sans-serif)

Шрифты загружаются через Google Fonts.
